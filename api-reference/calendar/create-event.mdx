## Create Calendar Event

Create a new event on the authenticated user's calendar.

### Authentication

<Note>Requires a valid user JWT token.</Note>

### Request Body

<ParamField body="title" type="string" required>
  Event title.
</ParamField>

<ParamField body="start_at" type="string" required>
  Start time in RFC 3339 format.
</ParamField>

<ParamField body="end_at" type="string" required>
  End time in RFC 3339 format.
</ParamField>

<ParamField body="all_day" type="boolean" required>
  Whether this is an all-day event.
</ParamField>

<ParamField body="description" type="string">
  Event description.
</ParamField>

<ParamField body="location" type="string">
  Event location.
</ParamField>

<ParamField body="attendees" type="array">
  Array of attendees with `email` and `display_name`.
</ParamField>

<ParamField body="recurrence" type="object">
  Recurrence rule with `frequency` (`"daily"`, `"weekly"`, `"monthly"`, `"yearly"`).
</ParamField>

### Example

```bash
curl -X POST https://api.ama2.me/api/v1/calendar/events \
  -H "Authorization: Bearer YOUR_JWT_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{
    "title": "Team standup",
    "start_at": "2026-04-10T09:00:00Z",
    "end_at": "2026-04-10T09:30:00Z",
    "all_day": false,
    "description": "Daily sync",
    "attendees": [
      {"email": "alice@example.com", "display_name": "Alice"}
    ]
  }'
```
