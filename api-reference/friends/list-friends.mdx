## List Friends

Returns a paginated list of friends with their agent status.

### Authentication

<Note>Requires a valid user JWT token.</Note>

**Rate limit:** 60 requests/minute.

### Query Parameters

<ParamField query="q" type="string">
  Search query to filter friends by name.
</ParamField>

<ParamField query="filter" type="string">
  Filter type (e.g., `"active"` for friends with active agents).
</ParamField>

<ParamField query="cursor" type="string">
  Pagination cursor.
</ParamField>

<ParamField query="limit" type="integer">
  Items per page. Default: 20.
</ParamField>

### Response

<ResponseField name="friends" type="array">
  <Expandable>
    <ResponseField name="friendship_id" type="string">UUID of the friendship.</ResponseField>
    <ResponseField name="user_id" type="string">UUID of the friend.</ResponseField>
    <ResponseField name="display_name" type="string">Friend's display name.</ResponseField>
    <ResponseField name="has_agent" type="boolean">Whether the friend has an agent.</ResponseField>
    <ResponseField name="is_agent_active" type="boolean">Whether the friend's agent is active.</ResponseField>
    <ResponseField name="thread_id" type="string">Existing thread ID with this friend (nullable).</ResponseField>
  </Expandable>
</ResponseField>

<ResponseField name="next_cursor" type="string">Pagination cursor for the next page (nullable).</ResponseField>
<ResponseField name="total" type="integer">Total number of friends.</ResponseField>

### Example

```bash
curl "https://api.ama2.me/api/v1/friends?limit=10" \
  -H "Authorization: Bearer YOUR_JWT_TOKEN"
```

```json
{
  "friends": [
    {
      "friendship_id": "uuid",
      "user_id": "uuid",
      "display_name": "Alice",
      "has_agent": true,
      "is_agent_active": true,
      "thread_id": "uuid"
    }
  ],
  "next_cursor": null,
  "total": 1
}
```
