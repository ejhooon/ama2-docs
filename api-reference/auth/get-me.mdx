## Get Current User

Returns the profile of the currently authenticated user, including their agent link if one exists.

### Authentication

<Note>Requires a valid user JWT token.</Note>

### Response

<ResponseField name="user" type="object">
  <Expandable>
    <ResponseField name="id" type="string">UUID of the user.</ResponseField>
    <ResponseField name="email" type="string">Email address.</ResponseField>
    <ResponseField name="username" type="string">Unique username.</ResponseField>
    <ResponseField name="display_name" type="string">Display name.</ResponseField>
    <ResponseField name="avatar_url" type="string">Profile image URL (nullable).</ResponseField>
    <ResponseField name="plan_type" type="string">Subscription tier: `"basic"` or `"starter"`.</ResponseField>
    <ResponseField name="trial_ends_at" type="string">Trial expiration timestamp (nullable).</ResponseField>
    <ResponseField name="email_notifications_enabled" type="boolean">Email notification preference.</ResponseField>
  </Expandable>
</ResponseField>

<ResponseField name="agent_link" type="object">
  The user's agent profile (nullable if no agent created).
</ResponseField>

### Example

```bash
curl https://api.ama2.me/api/v1/auth/me \
  -H "Authorization: Bearer YOUR_JWT_TOKEN"
```

```json
{
  "user": {
    "id": "uuid",
    "email": "alice@example.com",
    "username": "alice",
    "display_name": "Alice",
    "avatar_url": null,
    "plan_type": "starter",
    "trial_ends_at": null,
    "email_notifications_enabled": true
  },
  "agent_link": {
    "id": "uuid",
    "custom_slug": "alice-agent",
    "display_name": "Alice's Agent",
    "is_active": true
  }
}
```
