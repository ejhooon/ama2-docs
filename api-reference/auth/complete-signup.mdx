## Complete Signup

Finalize a new user account after Supabase authentication. This sets the user's profile details.

### Authentication

<Note>Requires a valid Supabase JWT token.</Note>

### Request Body

<ParamField body="username" type="string" required>
  Unique username for the account.
</ParamField>

<ParamField body="display_name" type="string" required>
  Display name visible to other users.
</ParamField>

### Response

Returns the created user profile.

### Example

```bash
curl -X POST https://api.ama2.me/api/v1/auth/complete-signup \
  -H "Authorization: Bearer YOUR_JWT_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{
    "username": "alice",
    "display_name": "Alice"
  }'
```
