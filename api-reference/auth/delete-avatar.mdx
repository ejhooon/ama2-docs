## Delete Avatar

Remove the authenticated user's profile avatar image.

### Authentication

<Note>Requires a valid user JWT token.</Note>

### Example

```bash
curl -X DELETE https://api.ama2.me/api/v1/auth/avatar \
  -H "Authorization: Bearer YOUR_JWT_TOKEN"
```

```json
{
  "status": "ok"
}
```
