## Check Slug Availability

Check whether an agent slug is available for use.

<Note>This endpoint is public and does not require authentication.</Note>

### Path Parameters

<ParamField path="slug" type="string" required>
  The slug to check. Must be URL-safe (lowercase alphanumeric and hyphens).
</ParamField>

### Response

<ResponseField name="available" type="boolean">
  Whether the slug is available.
</ResponseField>

<ResponseField name="reserved" type="boolean">
  Whether the slug is a reserved word.
</ResponseField>

### Example

```bash
curl https://api.ama2.me/api/v1/auth/check-slug/my-agent
```

```json
{
  "available": true,
  "reserved": false
}
```
