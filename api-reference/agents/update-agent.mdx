## Update Agent Config

Update the agent link (profile) settings.

### Authentication

<Note>Requires a valid user JWT token. Starter plan required.</Note>

### Request Body

<ParamField body="custom_slug" type="string">
  URL slug for the public page. Must be URL-safe.
</ParamField>

<ParamField body="display_name" type="string">
  Agent display name.
</ParamField>

<ParamField body="agent_name" type="string">
  AI system name.
</ParamField>

<ParamField body="welcome_message" type="string">
  Greeting message for visitors.
</ParamField>

<ParamField body="is_active" type="boolean">
  Whether the agent is live.
</ParamField>

<ParamField body="creator_bio" type="string">
  Owner's bio text.
</ParamField>

<ParamField body="agent_instructions" type="string">
  Custom instructions for the agent's behavior.
</ParamField>

### Example

```bash
curl -X PUT https://api.ama2.me/api/v1/owner/agent \
  -H "Authorization: Bearer YOUR_JWT_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{
    "display_name": "Updated Agent Name",
    "welcome_message": "Welcome! Ask me anything.",
    "is_active": true
  }'
```
