## Check Agent Slug

Check whether a specific slug is available for use as an agent profile URL.

<Note>This endpoint is public and does not require authentication.</Note>

### Request Body

<ParamField body="slug" type="string" required>
  The slug to check availability for.
</ParamField>

### Response

<ResponseField name="available" type="boolean">Whether the slug is available.</ResponseField>

### Example

```bash
curl -X POST https://api.ama2.me/api/v1/owner/agent/check-slug \
  -H "Content-Type: application/json" \
  -d '{"slug": "my-agent"}'
```

```json
{
  "available": true
}
```
