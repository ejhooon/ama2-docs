## Remove Participant

Remove a participant from an existing thread.

### Authentication

<Note>Requires a valid user JWT token with `can_manage_participants` capability.</Note>

### Path Parameters

<ParamField path="thread_id" type="string" required>
  UUID of the thread.
</ParamField>

<ParamField path="user_id" type="string" required>
  UUID of the user to remove.
</ParamField>

### Example

```bash
curl -X DELETE https://api.ama2.me/api/v1/chat/threads/{thread_id}/participants/{user_id} \
  -H "Authorization: Bearer YOUR_JWT_TOKEN"
```

```json
{
  "status": "ok"
}
```
