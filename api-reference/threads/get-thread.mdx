## Get Thread

Returns detailed information about a specific thread.

### Authentication

<Note>Requires a valid user JWT token. User must be a participant in the thread.</Note>

### Path Parameters

<ParamField path="thread_id" type="string" required>
  UUID of the thread.
</ParamField>

### Response

<ResponseField name="thread_id" type="string">UUID of the thread.</ResponseField>
<ResponseField name="is_archived" type="boolean">Whether the thread is archived.</ResponseField>
<ResponseField name="participants" type="array">List of thread participants with their roles and capabilities.</ResponseField>
<ResponseField name="last_message_at" type="string">Timestamp of the last message.</ResponseField>
<ResponseField name="unread_count" type="integer">Number of unread messages.</ResponseField>
<ResponseField name="needs_attention" type="boolean">Whether the thread needs owner attention.</ResponseField>

### Example

```bash
curl https://api.ama2.me/api/v1/chat/threads/{thread_id} \
  -H "Authorization: Bearer YOUR_JWT_TOKEN"
```
