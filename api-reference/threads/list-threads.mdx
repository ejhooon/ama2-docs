## List Threads

Returns all threads the authenticated user participates in.

### Authentication

<Note>Requires a valid user JWT token.</Note>

### Response

Returns an array of thread objects:

<ResponseField name="threads" type="array">
  <Expandable>
    <ResponseField name="thread_id" type="string">UUID of the thread.</ResponseField>
    <ResponseField name="is_archived" type="boolean">Whether the thread is archived.</ResponseField>
    <ResponseField name="participants" type="array">List of thread participants.</ResponseField>
    <ResponseField name="last_message_at" type="string">Timestamp of the last message.</ResponseField>
    <ResponseField name="last_message_preview" type="string">Preview of the last message.</ResponseField>
    <ResponseField name="unread_count" type="integer">Number of unread messages.</ResponseField>
  </Expandable>
</ResponseField>

### Example

```bash
curl https://api.ama2.me/api/v1/chat/threads \
  -H "Authorization: Bearer YOUR_JWT_TOKEN"
```
