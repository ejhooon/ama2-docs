## Add Participant

Add a new participant to an existing thread. Requires `can_manage_participants` capability.

### Authentication

<Note>Requires a valid user JWT token with `can_manage_participants` capability.</Note>

### Path Parameters

<ParamField path="thread_id" type="string" required>
  UUID of the thread.
</ParamField>

### Request Body

<ParamField body="user_id" type="string" required>
  UUID of the user to add.
</ParamField>

### Example

```bash
curl -X POST https://api.ama2.me/api/v1/chat/threads/{thread_id}/participants \
  -H "Authorization: Bearer YOUR_JWT_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{
    "user_id": "target-user-uuid"
  }'
```
