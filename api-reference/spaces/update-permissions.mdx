## Update Space Permissions

Change the permission preset for a space.

### Authentication

<Note>Requires a valid user JWT token. Must be the space owner.</Note>

### Path Parameters

<ParamField path="space_id" type="string" required>
  UUID of the space.
</ParamField>

### Request Body

<ParamField body="permission_preset" type="string" required>
  Permission preset to apply.
</ParamField>

### Example

```bash
curl -X PATCH https://api.ama2.me/api/v1/spaces/{space_id}/permissions \
  -H "Authorization: Bearer YOUR_JWT_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{"permission_preset": "public"}'
```
